Functions➖

A relation R: A →\to→﻿ B is considered as function if

(i) Every element in A has mapping

(ii) Every element in A has unique mapping

Vertical Line test

A graphical method to check if the curve is function or not , Draw vertical line at different positions if it intersects the curve at more than one point , then it is not a function.

Domain and Range

  • Every machine has some limitation in accepting inputs.
  • The set of all acceptable input of a machine F is called its domain.
  • The set of all resulting outputs is called its range.
