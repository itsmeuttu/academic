  1. Using the factor:

    If the quadratic equation is factorable, you can use factoring to find the solutions. This method involves rewriting the equation in the form of two binomials equal to zero and then solving for x. For example, given the equation x^2 + 5x + 6 = 0, you can factor it as (x + 2)(x + 3) = 0 and set each binomial equal to zero to find the solutions.
